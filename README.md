# 🧮 Calculator App (PyQt6)

A modern calculator built with **PyQt6**, designed with clean architecture and modular structure.

## 🚧 Status
This project is currently under active development.

## 🚀 Features

* Keyboard support
* Button-based input
* Expression validation
* Clean UI
* Modular architecture (UI / Core)

## 🧠 Architecture

* `ui/` → User Interface (PyQt6 components)
* `core/` → Application logic (operations, processing)
* `utils/` → Helper functions (validators, tools)

## 🛠️ Installation

```bash
git clone https://github.com/your-username/calculator-app.git
cd calculator-app
pip install -r requirements.txt
python main.py
```

## 📦 Requirements

* Python 3.10+
* PyQt6
